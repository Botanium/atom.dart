# atom

[![Build Status](https://travis-ci.org/dart-atom/atom.dart.svg)](https://travis-ci.org/dart-atom/atom.dart)

Atom APIs for Dart.

## Usage

This library is designed to be compiled with DDC and use the new `package:js`
interop.
